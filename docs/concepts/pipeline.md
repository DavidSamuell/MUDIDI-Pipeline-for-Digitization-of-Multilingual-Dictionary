# Pipeline concepts

```text
page image/PDF
    ↓
Stage 1 faithful transcription
    ↓
Stage 2 Pass 1 parse-rule discovery
    ↓
Stage 2 Pass 2 MDF extraction
    ↓
evaluation and language-script reports
```

Stage 1 preserves visible text. Stage 2 structures that transcript without treating the page image as an alternate text source.
